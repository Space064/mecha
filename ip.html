Főoldal DNS Források

Mik azok az IP címek és hogyan működnek?

Az IP (Internet Protocol) címek hasonlítanak a telefonszámokhoz, arra szükségesek hogy ezközeink tudjanak kommunikálni egymással és az Internettel.
Egy átlagos otthoni hálozatban mindegyik eszköz IP címe hasonlóan néz ki. Hogy megnézd saját IP címedet (Windows) nyisd meg a Parancssort, és írd be ezt: ipconfig
Valami hasonlót fogsz látni.
Az IPv4 Adderss a saját eszközöd IP címe.

Miért így néz ki?
Ez a Subnet Mask-nak köszönhető mivel az első három oktett le van fixálva és csak az utolsó változhat 0 és 255 között. Az első 2 oktett privát (vagy helyi) IP címeknél általában mindig 192.168.-al kezdődik.

És mi az a Default Gateway?
A Default Gateway másnéven a Router osztja ki az eszközöknek az IP címeket. A Router utolsó oktettje általában mindig 1.
Egy átlagos otthoni hálózat a képen látható módon van kialakítva. Ha az Internetet akarjuk elérni akkor szükségünk lesz egy nyilvános IP címre is. Ezt az Internetszolgáltató adja, de nem minden egyes eszköznek a háztartáson belül, hanem csak a Routernek. Ezért az Internetet csakis a Routeren keresztül érhetjük el.
Hogyan érem el a Google keresőt?

A számítógép küld egy kérelmet a Router felé hogy te a google.com-ot akarod elérni. A Router nem tudja hol találja Google barátunkat és megkerdezi a DNS-t. A DNS tudja a Google IP címét és átirányít arra. Megérkeztünk a google.com weboldalra.
Érdekesség:
Az iskola helyi hálózati kiépítése valószínűleg a következő.

Ahelyett hogy több 100 gépet 1 Routerbe kapcsolánk úgynevezett Switch-ek segítségével kapcsoljuk össze a gépeket, szervereket, Wi-Fi erősítőket az iskola egyes pontjain. Ezeket a Switch-eket láthatod a folyósókon a falra rögzítve, vagy egyes termekben.

Összefoglalva

- Az IP címek segítségével kommunikálnak az eszközeink.
- Vannak privát/helyi IP címek és nyilvánosak is.
- Nyilvános IP címek szükségesek az Internet eléréséhez.
- Háztartásonként 1 nyilvános IP cím áll rendelkezésre a Routeren keresztül.
- A Router feladata hogy a Helyi Hálózaton lévő eszközöket az Internettel öszekapcsolja.
